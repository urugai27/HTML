ぶながーとは



ぶながー(bunagaa)とは、赤い髪の毛、赤い身体、そして身長は１ｍ２０ｃｍ
ぐらい、川の蟹を食べているの目撃された。場所は沖縄県国頭郡大宜味村のと
ある村僕の隣近所に住んでいる爺さんから、聞いた話です。
また、友達の話（信憑性は・・・）では夜中か、ガラス越しに何か赤い人らし
きものが、部屋をのぞいていたらしい
そういえば、僕らの村々では、毎年夏（多分夏休みの頃）、ぶながーホイホイ
といって爆竹をならしていた。

ぶながーが苦労した TuxGuitarの使い方



■一時停止、再開機能がない（一時停止が停止と同じ機能、バグ？）

一時停止が、停止と同じ機能。曲の途中で一時停止し、停止したあとの続きから再開する
ことができない。
Microsoftが提供するProcessExplorerを使えば、そのアプリでタックスギターの
演奏（プロセス）を一時停止(Suspend)し、再開(Resume)することができる。
　
1.拍子 ⇒　
①拍子記号 4/4を確認する
②テンポ 103に設定
③調子記号 ♭が６つを選択 (変ト長調)
ド（Ｃ）は譜面通りの音符を位置付けできない！
シ（Ｂ）ナチュラル?のシで代替する


2.小節の編集

※上手く音符を設定出来なかった場合は、それにかかる拍子を１泊削除する
音符／エフェクト ⇒ 1拍削除(Cntl + Delete)
すると1拍の休符が割り当てられる

①編集したい音価の一番小さいサイズをスールボックスで選択
八音符を選択すると、八音符の休符が４拍分（八音符の休符が８個）
②下段の音符から設定する （上段は下段につられる）

※５線符のド（Ｃ）をマウスでクリックしたいが、出来ない！
仕方ないのでシ（Ｂ）にクリックしナチュラル?のシで代替する

そして、音符／エフェクト ⇒ 上弦にシフト(shift+ Up)
または 下弦にシフト(shift+ Down)

なお、上下へのシフトが出来るのは〇〇制限がある

3.歌詞の編集 二つの方法がある
ネットからダウンロードできる演奏データでは、小節で位置づけ
ができるテキスト挿入を利用している。

①テキスト挿入（小節の上段に表示）
音符／エフェクト ⇒ テキスト挿入
文字が表示される開始位置は下▽上△

②歌詞エディッタ（小節の下段に表示）
トラック ⇒ 歌詞エディッタ
文字が表示される開始位置は歌詞エディッタ画面で設定した開始小節

その他 これは

Edit Select Voice 1 音符の旗が上向き　
Edit Select Voice 2 音符の旗が下向き　 休符がうめられない！
